>
>

>
# Instalação do SSH Server #

> 
A sequencia de comandos abaixo autualiza os pacotes do Linux e em seguida instala o SSH Server.
>

```
$ sudo apt update
$ sudo apt upgrade
$ sudo apt install openssh-server
```
